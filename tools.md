# Typiske værktøjer

## Her er en liste med typiske væktøjer:

- **Nmap:** A network scanner used to discover hosts and services on a computer network by sending packets and analyzing the responses.

- **Metasploit:** A penetration testing framework that enables the development, testing, and execution of exploit code against a remote target machine.

- **Netcat (nc):** Often dubbed the Swiss-army knife for TCP/IP, it's used for reading from and writing to network connections using TCP or UDP.

- **Wireshark:** A network protocol analyzer that lets you capture and interactively browse the traffic running on a computer network.

- **Burp Suite**: An integrated platform for performing security testing of web applications. It contains a variety of tools with powerful interfaces.

- **John the Ripper**: A fast password cracker, currently available for many flavors of Unix, Windows, DOS, and OpenVMS. It's used to detect weak Unix passwords.

- **Hydra:** A very fast network logon cracker which support many different services. It's used for brute-forcing login forms, FTP servers, SSH, and more.

- **SQLmap:** An open-source penetration testing tool that automates the process of detecting and exploiting SQL injection flaws and taking over database servers.

- **DirBuster/**Dirsearch/Gobuster: Tools for directory enumeration. They are used to discover web content that is not linked from the visible content that you can browse to in a web browser.

- **Nikto:** A web server scanner which performs comprehensive tests against web servers for multiple items, including over 6700 potentially dangerous files/programs.

- **OWASP ZAP (Zed Attack Proxy):** An open-source web application security scanner. It's designed to find security vulnerabilities in web applications.

- **Enum4linux:** A tool used to enumerate information from Windows and Samba systems. It attempts to offer similar functionality to enum.exe from Sysinternals.

- **LinEnum/PEASS**- Privilege Escalation Awesome Scripts SUITE (like LinPEAS, WinPEAS): Scripts that enumerate information and perform automated checks to identify possible paths to escalate privileges on Unix/Linux and Windows systems.

- **Gobuster:** A tool used to brute-force URIs (directories and files) in web sites and DNS subdomains (with wildcard support).

- **Steghide:** A steganography program that is able to hide data in various kinds of image- and audio-files. It's used in challenges related to hiding information inside files.

- **ExifTool:** A platform-independent Perl library plus a command-line application for reading, writing, and editing meta information in a wide variety of files. It's particularly useful for extracting metadata from images and documents.

- **Hashcat:** An advanced password recovery tool. It's known for its speed and versatility in cracking hashed passwords.
