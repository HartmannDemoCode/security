# Værktøjer i katerogi

- Husk på at recon og scanning ikke er det samme. Til trods for dette kan NMAP godt betragtes som både recon & scanning. Denne liste er ikke udtømmende men er udelukkende en hjælp til Jer. I kan selv søge disse lister frem på internettet hvis I ønsker. 
## Reconnaissance (Recon)

- Tools and resources used for gathering information about the target system, network, or organization.

- Nmap: Identifies open ports, services, and system details.
- Netcat (nc): Gathers information by establishing connections to ports.

## Scanning

- Tools used for more detailed analysis of the discovered services and vulnerabilities.

- Nmap: For detailed scanning of open ports and services.
- Nikto: Web server scanner which performs comprehensive tests against web servers for multiple items.
- Gobuster/DirBuster/Dirsearch: Tools for brute-forcing directories and file names on web and application servers.
- OWASP ZAP (Zed Attack Proxy): Finds vulnerabilities in web applications.
- Burp Suite: Analyzes and attacks web applications.
- WPScan: Identifies vulnerabilities in WordPress sites.
- SQLmap: Automates the detection and exploitation of SQL injection flaws.

## Exploiting

- Tools and resources used for exploiting vulnerabilities, gaining access, and privilege escalation.

- Metasploit: Exploits vulnerabilities and gains access to remote systems.
- Hydra: Performs brute-force attacks on various protocols to gain unauthorized access.
- John the Ripper: Password cracking for gaining access.
- Hashcat: Advanced password recovery.
- MSFvenom: Creates custom payloads for exploiting vulnerabilities.
- LinEnum/PEASS - Privilege Escalation Awesome Scripts SUITE (like LinPEAS, WinPEAS): Scripts that enumerate information and perform automated checks to identify possible paths to escalate privileges.
- GTFOBins: A curated list of Unix binaries that can be used to bypass local security restrictions in misconfigured systems.
